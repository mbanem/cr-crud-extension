Create CRUD Support

To create a UI Form for CRUD operations against the underlying ORM
fill out the Candidate Fields by entering field names in the Field 
Name input box pressing the Enter key or expand a table from the 
Select Fields from ORM block and click on a field name avoiding the
auto-generating fields usually colored in pink.
The UI Form +page.svelte with accompanying +page.server.ts will be 
created in the route specified in the Route Name input box.
